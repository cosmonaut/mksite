mksite - static website generator

mksite is a simple static website generator that uses markdown and a linux directory structure to generate and maintain a website. It is designed to be simple and convenient. The main idea is to be able to build a website simply by creating directories and markdown files in linux after which the output can be rsynced to the server.

This code was originally copied from genosite at suckless.org (see: http://hg.suckless.org/genosite/) and uses some of the ideas from poole (see: http://bitbucket.org/obensonne/poole/src). If you need a more robust website generator consider hyde (http://ringce.com/hyde).

This site was built using mksite and serves as an example of what the default output should look like.
